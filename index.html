Web Programming Showcase

Website ini adalah sebuah showcase yang membanggakan dari prestasi dan kreativitas para peserta Tech Muda II. About Tech Muda II

Wujudkan Perempuan Muda Terampil Teknologi

Tech Muda II adalah program pengembangan yang mendukung kaum muda perempuan, termasuk penyandang disabilitas, melalui pelatihan di bidang Teknologi Informasi yaitu pelatihan web programming, digital marketing, dan soft skills.

Tech Muda II memperoleh dukungan berupa pelatihan, mentoring, pembelajaran berbasis kerja (magang) atau pembelajaran berbasis proyek, dan dukungan penempatan kerja
